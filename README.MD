### 服务端支持

- [x] Spigot 1.8+
- [ ] Spigot Legacy (1.7-)
- [x] Bungeecord 理论全版本

### 命令列表

- noir
    - help - 获取命令帮助
    - reload - 重载插件配置文件
    - version - 查看运行信息
    - tasks - 查看群命令队列
